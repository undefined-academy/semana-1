## Comandos basicos de terminal

| Nombre del comando | Descripcion                                                        |
| -----------------: | ------------------------------------------------------------------ |
|                 ls | Lista los directorios y archivos en la ruta donde estemos ubicados |

|                 cd | Cambia entre directorios                                           |
