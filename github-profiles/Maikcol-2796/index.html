My projects

Tic-tac-toe

The project is a tic tac toe game created using React JS and CSS, with options to play against the computer or another player locally. It includes features to save the user's score and the last unfinished game. The main goal is to provide an enjoyable and visually appealing gaming experience, while also allowing users to easily track their progress over time.

School enrollment management system

The project aims to develop a document management system for student registration using Bootstrap for the user interface design, React JS for the frontend, Python as the main programming language, and FastAPI as the framework for developing the RESTful API. The goal of the project is to simplify and streamline the student registration process, allowing them to easily and quickly upload the required documents, and providing system administrators with an efficient tool to manage and approve the received documents.

Other projects with React

This project repository contains multiple React JS projects that showcase the use of different React hooks such as useState, useRef, useEffect, useMemo, and useContext. These hooks are used for managing state, handling side effects, optimizing performance, and sharing data between components. The goal of the repository is to help developers improve their React JS skills by demonstrating practical use cases for these hooks.
