HTML en unos pocos minutos

Mi perfil de github

1. Estructura Básica:

        
        <!DOCTYPE html>
        <html lang="es">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Título de la Página</title>
        </head>
        <body>
    
        <!-- Contenido de la página va aquí -->
    
        </body>
        </html>

  • !DOCTYPE html: Define la versión de HTML que estás utilizando.
  • html: Elemento raíz de la página.
  • head: Contiene meta-información sobre el documento.
  • meta charset="UTF-8":: Especifica la codificación de caracteres.
  • meta name="viewport" content="width=device-width, initial-scale=1.0": Configura la visualización en dispositivos móviles.
  • title: Define el título de la página.
  • body: Contiene el contenido visible de la página.

2. Elementos Básicos

Encabezados:

        
        <h1>Encabezado 1</h1>
        <h2>Encabezado 2</h2>
        <!-- ... hasta h6 -->

Párrafos:

        
        <p>Este es un párrafo.</p>

Listas:

Listas no ordenadas:

        
        <ul>
            <li>Elemento 1</li>
            <li>Elemento 2</li>
        </ul>

Listas ordenadas:

        
        <ol>
            <li>Elemento 1</li>
            <li>Elemento 2</li>
        </ol>

Enlaces:

        
        <a href="https://www.ejemplo.com">Enlace a Ejemplo</a>

Imágenes:

        
        <img src="imagen.jpg" alt="Descripción de la imagen">

3. Formularios:

        
        <form action="/procesar" method="post">
            <label for="nombre">Nombre:</label>
            <input type="text" id="nombre" name="nombre" required>

            <label for="email">Correo Electrónico:</label>
            <input type="email" id="email" name="email" required>

            <input type="submit" value="Enviar">
        </form>

4. Estructuras semánticas

Secciones:

        
        <section>
            <h2>Sección 1</h2>
            <p>Contenido de la sección 1.</p>
        </section>
    
        <section>
            <h2>Sección 2</h2>
            <p>Contenido de la sección 2.</p>
        </section>

Artículos:

        
        <article>
            <h2>Artículo 1</h2>
            <p>Contenido del artículo 1.</p>
        </article>

        <article>
            <h2>Artículo 2</h2>
            <p>Contenido del artículo 2.</p>
        </article>
