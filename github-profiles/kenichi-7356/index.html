Alias creados para usar en un CLI linux

Dependiendo del tipo de trabajo que realice en su sistema Linux, es posible que deba ingresar los mismos comandos largos y complicados con frecuencia. El comando alias le permite crear accesos directos para estos comandos, lo que facilita su recuerdo y uso.

* Tomado de "https://phoenixnap.com/kb/linux-alias-command"

Alias Comando
bootcamp cd ~/Code/el-mundo-de-markdown
actualizar sudo apt-get update && sudo apt upgrade
status git status

Comando que mas uso en la terminal

Comando Descripcion
ls lista los directorios en el cual este ubicado en el CLI.
pwd Muestra en el CLI la ubicación en la cual se encuentra ubicado.
cd Se usa para navegar entre directorio, combinado con ".." se devuelve un directorio, "/" va al directorio raiz, entre otros
clear usado para limpiar la pantalla de la Terminal
