# 5 comandos mas usados

| comando | descripcion                                                          |
| ------- | -------------------------------------------------------------------- |
| ls      | lista el contenido de un directorio                                  |
| cd      | cambia de directorio                                                 |
| rm      | es utilizado para eliminar archivos y directorios en SO Unix o Linux |
| mv      | es utilizado para mover o renombrar archivos o directorios           |
