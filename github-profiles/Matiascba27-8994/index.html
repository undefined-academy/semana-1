💻 Lista de comandos de la terminal de linux

  • ls: Lista los archivos de la carpeta actual.
  • cd: Nos permite navegar entre carpetas.
  • pwd: Nos nuestra la ruta absoluta del directorio actual.
  • mkdir: Crea una nueva carpeta dado un nombre para esa carpeta.
  • touch: Crea un archivo nuevo dada una ruta y/o nombre.

🎢 Atajos de alias

  1. ll(ls -l): Lista los archivos con detalles.
  2. update: Sudo apt update && sudo apt upgrade: para actualizar el sistema con apt.
