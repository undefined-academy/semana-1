## `üî• Shortcut`

```javascript
{
    "key": "ctrl+shift+l",
    "command": "editor.action.insertSnippet",
    "when": "editorTextFocus",
    "args": {
      "snippet": "console.log('üî• ${RELATIVE_FILEPATH} > line: ${TM_LINE_NUMBER} > ${TM_SELECTED_TEXT}:$2', ${TM_SELECTED_TEXT}$1);"
    },
  },
```

```javascript
    {
        "key": "ctrl+j",

        "when": "editorTextFocus",
        "args": {
          "snippet": "console.log(JSON.stringify({ ${TM_SELECTED_TEXT} }, null, 2));"

      },
```
