Terminal Commands

My favourite Terminal commands:

    
      

        
          

            Command
            Description
          

        
        
          

            ls-la
            Lists the directory elements, even the hidden ones
          

          

            mkdir
            Creates a new directory
          

          

            touch
            Creates a new file
          

          

            rm -i
            Removes a file asking before do it
          

          

            rm -r
            Removes a directory
          

          

            pwd
            Prints the actual route
          

          

            code .
            Opens a new VScode session with the actual file
          

          

            clear
            Cleans the content printed in the screen
