Practical Alias

Sometimes we want to save time when coding, so, we can asign some short alias to the commands in order to do it, there are some of my own:

Alias Command
cl: 'clear' Cleans the screen
findir: 'ls ~/ \| grep $1' Find a coincident directory
run: 'npm run dev' Runs a development server in local host in Vite
start: 'npm run start' Runs a development server in local host in JS and CRA
