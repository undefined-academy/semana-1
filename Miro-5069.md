# PASOS QUE SEGUI PARA HACER LA PRACTICA 2

1. Para hacer un fork del repositorio https://github.com/undefined-academy/semana1 se debe hacer clic en el botón "Fork" .

2. Luego, para clonar el repositorio forked en tu carpeta de HOME, se puede abrir una terminal en el sistema operativo y ejecutar el siguiente comando:
   #  `git clone https://github.com/<tu_usuario_github>/semana1.git`
     
3. Después, se debe crear un archivo markdown en la carpeta "githubprofiles" con el nombre de usuario de Discord, incluyendo el tag numérico, y con la extensión ".md". Por ejemplo: "mi_usuario_discord#1234.md". 

4. Dentro de ese archivo se puede agregar el enlace al perfil de GitHub utilizando la sintaxis de Markdown para enlaces.

5. Luego, se debe hacer commit de dicho archivo utilizando los comandos de git, como:
  ---------------------------------------------------------
  # `   git add githubprofiles/mi_usuario_discord-1234.md  `
 ----------------------------------------------------------
  # `    git commit -m "Agregando mi perfil de GitHub"  `
  ---------------------------------------------------------
  # `   git push origin main `  
    
6. Finalmente, se puede ir al repositorio del paso 1, hacer clic en el botón "New pull request", seleccionar el branch y los cambios que se quieren incorporar y agregar una descripción del Pull Request, explicando los cambios que se han realizado y cualquier otra información relevante. Luego, se puede hacer clic en el botón "Create Pull Request" para enviar la solicitud de incorporación de los cambios.
